### Fiber-Optic 
SFP Transceiver (Small Form-Factor Pluggable) 

| Feature       | Single-Mode Fiber | Multimode Fiber   |
| ------------- | ----------------- | ----------------- |
| Core size     | ~9 µm             | 50 / 62.5 µm      |
| Light path    | Single            | Multiple          |
| light source  | laser             | led               |
| Distance      | Very long         | Short             |
| Cost          | Higher            | Lower             |
| Typical color | Yellow            | Orange / Aqua     |
| Common use    | WAN / Backbone    | LAN / Data center |

### utp vs fiber
|Feature特征|UTP (Copper)UTP（铜缆）|Fiber纤维|
|---|---|---|
|Signal type信号类型|Electrical电气|Light光|
|Max distance最大距离|100 m100米|Hundreds of m to km数百米至公里|
|EMI/RFI电磁干扰/射频干扰|Affected做作的|Immune免疫|
|Speed速度|Up to 10 Gbps最高可达 10 Gbps|10 Gbps+|
|Cost成本|Low低的|Higher更高|
|Installation安装|Easy简单的|More complex更复杂|
|Security安全|Can be tapped可以轻敲|Harder to tap更难敲击|
## Day 3 TCP/IP & OSI model
### tcp/ip
| TCP/IP LayerTCP/IP 层   | What It Does它的功能                                    | Common Protocols通用协议                               | Example Use Case示例用例                             |
| ---------------------- | --------------------------------------------------- | -------------------------------------------------- | ------------------------------------------------ |
| **Application应用**      | User-facing network services面向用户的网络服务               | HTTP, HTTPS, FTP, SMTP, POP3, IMAP, DNS, SNMP, SSH | Web browsing, email, file transfer网页浏览、电子邮件、文件传输 |
| **Transport运输**        | End-to-end delivery, ports, reliability端到端交付、港口、可靠性 | **TCP**, UDP                                       | TCP → web/email, UDP → DNS/streaming             |
| **Internet互联网**        | Logical addressing and routing逻辑寻址和路由               | **IP (IPv4/IPv6)**, ICMP, IPsec                    | Packet routing between networks网络间的数据包路由         |
| **Network Access网络访问** | Physical + data link delivery物理+数据链路交付              | Ethernet, ARP, Wi-Fi                               | Frame delivery on local network本地网络上的帧交付         |
